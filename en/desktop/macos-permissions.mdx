## Summary

macOS requires you to accept 4 permissions before kb2b Desktop can capture meetings:

| Permission | What for | How to grant |
|---|---|---|
| **Microphone** | Capture your voice in the meeting | Automatic prompt at startup |
| **Screen Recording** | Capture system audio (voices of other participants) | Automatic prompt + System Settings |
| **Accessibility** | Auto-detect Zoom/Meet/Teams meetings | System Settings → Privacy & Security → Accessibility |
| **Camera** | Detect active meeting window (does not record video) | Automatic prompt |

## If the prompt doesn't show

If kb2b Desktop doesn't show the prompt or you closed it by mistake, open System Settings and add kb2b Desktop manually. The app includes direct shortcuts to each pane:

- Microphone: `Settings → Privacy & Security → Microphone`
- Screen Recording: `Settings → Privacy & Security → Screen Recording`
- Accessibility: `Settings → Privacy & Security → Accessibility`
