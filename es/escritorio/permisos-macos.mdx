## Resumen

macOS requiere que aceptes 4 permisos antes de que kb2b Desktop pueda capturar reuniones:

| Permiso | Para qué | Cómo concederlo |
|---|---|---|
| **Micrófono** | Capturar tu voz en la reunión | Prompt automático al iniciar |
| **Grabación de pantalla** | Capturar audio del sistema (voces de los demás participantes) | Prompt automático + Ajustes del Sistema |
| **Accesibilidad** | Detectar automáticamente reuniones en Zoom/Meet/Teams | Ajustes del Sistema → Privacidad y Seguridad → Accesibilidad |
| **Cámara** | Detectar ventana de reunión activa (no graba vídeo) | Prompt automático |

## Si el prompt no aparece

Si kb2b Desktop no muestra el prompt o lo cerraste por error, abre Ajustes del Sistema y agrega kb2b Desktop manualmente. La app incluye atajos directos a cada panel:

- Micrófono: `Ajustes → Privacidad y Seguridad → Micrófono`
- Grabación de pantalla: `Ajustes → Privacidad y Seguridad → Grabación de pantalla`
- Accesibilidad: `Ajustes → Privacidad y Seguridad → Accesibilidad`
